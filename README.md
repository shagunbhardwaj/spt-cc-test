Credit Card Application
---------------------------

Technologies Used
1. Maven
2. JDK 11
3. Spring Boot
4. H2 embedded db
5. jUnit 5, Mockito
6. React

Steps to start
-----------------------------------------------
1. Server App Starts with CctestApplication.java file
2. Running npm start from "web" folder to start front end.

1. Have implemented one create and get apis.
